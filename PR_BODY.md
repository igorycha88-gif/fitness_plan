## Summary
Рабочая версия приложения v2.2 от 11.02.2026

## Изменения относительно develop (95 коммитов)

### Новые функции

**Параметры тела:**
- Ввод параметров тела в профиле (вес, обхват груди, талии, бицепса и т.д.)
- Графики и статистика параметров тела во вкладке Статистика
- Калькулятор расчета параметров тела
- Диалоги добавления измерений с валидацией

**Экран План:**
- Переработанный UI экрана планов
- Создание пользовательских планов тренировок
- Кнопка "Мой план" всегда доступна для создания плана
- Изоляция выполнения упражнений по дням тренировки

**Упражнения:**
- Картинки для 25+ упражнений (arnold_press, bench_press, lat_pulldown и др.)
- Отображение картинок на экранах ExerciseDetailScreen и ExerciseLibraryDetailScreen
- Прогресс выполнения упражнений с графиками
- Фильтр по упражнениям на экране статистики объёма
- Отметка выполнения упражнений без веса (кардио и растяжка)
- Расширенная библиотека упражнений

**Статистика:**
- График веса с цветовой схемой приложения
- График объёма тренировок с группировкой по дням/неделям
- Статистика по группам мышц с детализацией
- Фильтр статистики веса (ExposedDropdownMenuBox)
- Цветовая заливка дней по прогрессу выполнения

**Уведомления:**
- Напоминания о тренировках (WorkoutReminderScheduler, WorkoutReminderWorker)

**UI/UX:**
- Переработка экрана профиля с новым дизайном
- Новый экран входа с splash-изображением
- Выпадающие списки для выбора веса и повторений
- Автозаполнение полей веса и повторений рассчитанными значениями
- Адтивные layouts для разных размеров экрана

### Технические улучшения

- Обновлены тесты для совместимости с API 34/36
- Добавлены Hilt DI tests (TestModule, TestGreetingService)

- Удалены неиспользуемые файлы
