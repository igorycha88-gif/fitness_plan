# REQ-006: Оптимизация названий вкладок в разделе Статистика

**Status**: New
**Priority**: Medium
**Complexity**: Simple
**Created**: 2026-02-17
**Updated**: 2026-02-17

## Description
Оптимизировать названия вкладок на странице Статистика для улучшения UX. Название "Параметры тела" слишком длинное для tab layout на мобильных устройствах. Рассмотреть альтернативные варианты: короткое слово или использование иконок.

## User Story
Как пользователь,
Я хочу видеть компактные и понятные названия вкладок,
Чтобы легко переключаться между разделами статистики без горизонтального скролла.

## Current State Analysis

### Текущие вкладки:
| Индекс | Название | Длина (символов) | Проблема |
|--------|----------|------------------|----------|
| 0 | "Вес" | 3 | OK |
| 1 | "Прогресс" | 8 | OK |
| 2 | "Параметры тела" | 15 | Слишком длинное |

### Расположение в коде:
- `StatisticsScreen.kt:16` - определение вкладок:
  ```kotlin
  val tabs = listOf("Вес", "Прогресс", "Параметры тела")
  ```

### Проблемы текущего решения:
1. "Параметры тела" (15 символов) не помещается на узких экранах
2. Текст обрезается или вызывает горизонтальный скролл
3. Несбалансированная ширина вкладок визуально неэстетична

## Proposed Solution

### Вариант A: Переименование в короткое слово (РЕКОМЕНДУЕТСЯ)

**Новое название: "Замеры"**
- Длина: 6 символов (было 15)
- Сохраняет смысл: пользователь понимает, что это замеры параметров тела
- Баланс с другими вкладками: Вес(3), Прогресс(8), Замеры(6)

**Альтернативные варианты:**
| Вариант | Длина | Описание | Рейтинг |
|---------|-------|----------|---------|
| "Замеры" | 6 | Наиболее точное, общепринятое | ⭐⭐⭐⭐⭐ |
| "Тело" | 4 | Короткое, но менее специфичное | ⭐⭐⭐⭐ |
| "Данные" | 6 | Универсальное, но менее понятное | ⭐⭐⭐ |
| "Антропо" | 7 | Профессиональное, но непонятное | ⭐⭐ |

### Вариант B: Использование иконок (АЛЬТЕРНАТИВА)

**Замена всех вкладок на иконки:**
| Текущее | Иконка | Описание |
|---------|--------|----------|
| "Вес" | `Icons.Default.MonitorWeight` | Иконка весов |
| "Прогресс" | `Icons.Default.TrendingUp` | График роста |
| "Параметры тела" | `Icons.Default.Accessibility` | Человеческая фигура |

**Преимущества иконок:**
- Компактность для любого экрана
- Визуальная интуитивность
- Международность (без языка)

**Недостатки иконок:**
- Требует обучения пользователей
- Менее очевидно для новых пользователей
- Может потребовать tooltip/подсказки

### Рекомендация: Вариант A ("Замеры")
- Минимальные изменения в коде
- Сохраняется понятность для пользователей
- Не требует переработки дизайна

### Components Affected (Вариант B):

**StatisticsScreen.kt:**
```kotlin
// Было:
val tabs = listOf("Вес", "Прогресс", "Параметры тела")

// Станет:
data class TabItem(val icon: ImageVector, val label: String)

val tabs = listOf(
    TabItem(Icons.Default.MonitorWeight, "Вес"),
    TabItem(Icons.Default.TrendingUp, "Прогресс"),
    TabItem(Icons.Default.Accessibility, "Замеры")
)

// Tab component:
Tab(
    selected = selectedTab == index,
    onClick = { selectedTab = index },
    icon = { Icon(tab.icon, contentDescription = tab.label) },
    text = { Text(tab.label) } // опционально для accessibility
)
```
